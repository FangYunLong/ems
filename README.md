简易教务管理系统

项目二	    教务管理系统

软件环境	    JSP、Servlet、JavaScript、Ajax、Android

开发环境	    MyEclipse + Tomcat + MySQL

项目描述	    该项目是为管理员、教师和学生三种角色分别提供网页端、
手机端仅为教师与学生提供简易服务。为管理员提供系部、专业、班级、课程、学生、教师、账号管理功能。

责任描述	    参与了项目的功能划分的分析和设计的讨论，负责网页端的
专业、班级、课程、教师,手机端仅为教师与学生提供简易服务。

项目心得	    使用MVC模式开发，减少耦合；在设计数据库时，使用主外键，
将数据库表进行级联，减少冗余，建立视图，方便查询；使用ajax技术实现异步调用，
并在输入信息使用ajax和正则表达式检验数据的正确性；学习到手机端与服务器如何使用json数据交互。
